# Tic-Tac-Toe
This is my Tic-Tac-Toe game written on Python3 with Pygame library


## Requirements
To install pygame run:

```bash
    pip3 install pygame==2.0.1
```
or
```bash
    pip3 install requirements.txt
```

## Run
Run application:
```bash
    python3 tictactoe.py
```
